# DropQueue: Текущее состояние проекта

DropQueue — это платформа для управления музыкальной очередью в реальном времени для стримеров на Twitch.

## 1. Авторизация (Twitch OAuth)
- **Вход через Twitch**: Пользователи могут безопасно входить под своими аккаунтами.
- **Интеграция с БД**: Данные пользователей (имя, ID) сохраняются в PostgreSQL через Prisma.

## 2. Главная страница
- Брендированный лендинг в стиле Twitch ("Twitch Purple").
- Кнопки быстрого перехода к подаче треков или дашборду.

## 3. Навигация (Header)
- Отображение текущего пользователя.
- Ссылки на **Дашборд**, **Подачу трека** и **Зал славы**.

---

## 4. Основной функционал

### Подача треков (`/submit`)
- **Форма**: Возможность отправить название трека, ссылку (Spotify/YouTube/etc), описание и текст песни.
- **Статус в реальном времени**: Справа отображается блок, который через WebSockets показывает:
  - Что играет сейчас (Now Playing).
  - Результаты последних оценок (Recent Ratings).

### Дашборд стримера (`/dashboard`)
- **Очередь**: Список всех присланных треков в реальном времени.
- **Управление критериями**: Стример может сам добавлять/удалять пункты оценки (например, "Вайб", "Бит").
- **Панель оценки**: Слайдеры (1-10) для оценки текущего трека. При сохранении средний балл улетает всем зрителям.
- **Кнопка пропука (Skip)**: С обязательным указанием причины.
- **Завершение сессии**: Кнопка закрытия стрима с сохранением топ-треков в историю.

### Зал славы (`/leaderboard`)
- Глобальная таблица лидеров с лучшими треками за все время.

---

## 5. Техническая часть
- **WebSockets**: Custom Socket.io сервер.
- **Pause on Disconnect**: Автоматическая пауза сессии, если стример закрыл вкладку.
- **База данных**: Prisma 7 + PostgreSQL.
